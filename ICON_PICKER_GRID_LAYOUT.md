# 🎨 Палитра иконок Lucide в Filament

## Что добавлено

Настроен IconPicker для отображения иконок в виде **палитры (сетки)** вместо выпадающего списка.

## Настройки

### Layout: Grid

```php
IconPicker::make('icon')
    ->label('Иконка')
    ->sets(['lucide'])
    ->default('star')
    ->layout('grid')      // ← Отображение в виде палитры/сетки
    ->preload()           // ← Предзагрузка иконок для быстрого отображения
    ->columns(1)
```

## Дополнительные настройки

### Preload (Предзагрузка)
```php
->preload()
```
- Загружает все иконки заранее
- Быстрое отображение палитры
- Рекомендуется для grid layout

### Searchable (Поиск)
```php
->searchable()  // Включено по умолчанию
```
- Поиск иконок по названию
- Фильтрация в реальном времени

### Allow HTML
```php
->allowHtml()
```
- Разрешает HTML в иконках
- Нужно для кастомных SVG

### Icon Columns (Колонки в палитре)
```php
->iconColumns(8)  // Количество колонок в сетке
```
- По умолчанию: 8 колонок
- Можно изменить для большей/меньшей плотности

### Icon Size (Размер иконок)
```php
->iconSize('lg')  // sm, md, lg, xl
```
- Размер иконок в палитре
- По умолчанию: `md`

## Полная конфигурация

### Для EventResource (features)

```php
IconPicker::make('icon')
    ->label('Иконка')
    ->sets(['lucide'])              // Набор иконок
    ->default('star')                // Иконка по умолчанию
    ->layout('grid')                 // Палитра
    ->preload()                      // Предзагрузка
    ->searchable()                   // Поиск
    ->iconColumns(8)                 // 8 колонок
    ->iconSize('md')                 // Средний размер
    ->columns(1)                     // Ширина поля в форме
```

### Для TripResource (amenities)

```php
IconPicker::make('icon')
    ->label('Иконка')
    ->sets(['lucide'])
    ->default('wifi')
    ->layout('grid')
    ->preload()
    ->columns(1)
```

## Как это выглядит

### В Filament Admin Panel

1. **Открываете поле "Иконка"**
   - Сразу видна палитра со всеми иконками
   - Иконки расположены в сетке (grid)

2. **Поиск**
   - Вверху есть строка поиска
   - Вводите название: `wifi`, `star`, `map`
   - Палитра фильтруется в реальном времени

3. **Выбор**
   - Кликаете на иконку в палитре
   - Иконка выбирается
   - Поле закрывается

4. **Просмотр выбранной иконки**
   - Выбранная иконка отображается в поле
   - Можно изменить, кликнув снова
